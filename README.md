# LG Family

Private family operating system for calendar, groceries, reminders, tasks, family records, photos, vault items, and emergency information.

## Current status
This repository is being used to move the first UI prototype out of Hostinger Horizons and into GitHub so the app can be organized and built more deliberately.

## Initial product areas
- Dashboard
- Calendar
- Tasks
- Groceries
- Family
- Photos
- Vault
- Emergency
- Settings

## Short-term goals
1. Preserve the Horizons-generated layout.
2. Clean up the React/Vite structure.
3. Replace any platform-specific backend assumptions.
4. Add Supabase-backed auth, storage, and permissions later.
5. Keep sensitive family information separated behind strict access rules.

## Next build steps
- Finish importing the generated UI files.
- Refactor shared components.
- Add route-level organization.
- Define role-based access.
- Add backend schema and integrations.

## Planned stack
- React + Vite
- Tailwind CSS
- Supabase later for auth, database, storage, and permissions

## Notes
This repo should stay focused on private family use first. Commercialization can happen later if the product proves useful.
